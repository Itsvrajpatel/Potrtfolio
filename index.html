HTML

Hypertext Markup Language is a markup language used to create and structure content for the World Wide Web, consisting of a set of tags and attributes that define how a web page is displayed in a web browser.

CSS

Cascading Style Sheets is a styling language used to describe the visual appearance and layout of HTML documents, including elements such as fonts, colors, spacing, and positioning. It allows web developers to separate the presentation of content from its structure, making it easier to manage and update the appearance of a website.

JavaScript

JavaScript is a high-level, dynamic programming language used to create interactive and dynamic effects on web pages, as well as for server-side programming and development of desktop and mobile applications. It allows web developers to add functionality such as form validation, dynamic content updates, and animations to websites, as well as create complex web applications that can interact with web servers and databases.
